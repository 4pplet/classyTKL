# classyTKL

PCB for the classyTKL

## Some features:
- QMK & VIA
- Cherry PCB-stab support
- JST-connector for daughter board
- Polyfuse for overcurrent protection
- ESD-protection
- Reset-switch under spacebar
- ISO & ANSI-support
